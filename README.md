# Amazon Website Clone (HTML & CSS)

This is a front-end clone of Amazon's homepage, built using HTML & CSS as part of a beginner-friendly project.

## Features
- Structured layout similar to Amazon
- Product cards and footer sections
- Clean UI design
- Responsive design can be added in future using JavaScript
